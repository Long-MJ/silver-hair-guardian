# 银发守护者 (Silver Hair Guardian)

一款基于移动端H5页面的交互式防诈骗教育游戏，专为老年群体设计，通过模拟真实诈骗场景，提高老年人的防诈骗意识。

## 项目简介

"银发守护者"是一款针对老年群体的防诈骗警示教育游戏，通过模拟6种常见诈骗场景，让老年人在轻松的游戏环境中学习如何识别和应对诈骗。

### 主要功能

- **6个真实诈骗场景**：
  - 冒充公检法 - 假警察要求转账到"安全账户"
  - 养老投资 - 高回报理财骗局
  - 保健品诈骗 - 免费体检/假冒神药
  - 冒充亲友 - "我出事了快打钱"
  - 中奖退税 - "您中奖了"先交手续费
  - 黄昏恋诈骗 - 网络交友诱导投资

- **交互式游戏机制**：
  - 每场景3-5轮选择，每轮3个选项（2个错误，1个正确）
  - 错误选择提示，1秒后自动消失的弹窗
  - 重复错误选择惩罚机制
  - 胜利和失败界面，包含真实案例展示
  - 案例轮换机制，确保不重复直到全部轮换

- **老年友好设计**：
  - 大字体，清晰易读
  - 大按钮，便于点击
  - 温馨的界面风格，避免过于刺激的视觉元素
  - 简单直观的操作流程

## 技术实现

- **前端技术**：HTML5, CSS3, JavaScript
- **响应式设计**：适配各种移动设备屏幕
- **本地存储**：使用localStorage记录游戏状态
- **无需后端**：纯前端实现，可离线使用

## 如何使用

### 本地运行

1. 克隆或下载本项目
2. 在项目根目录启动本地服务器
   ```bash
   # 使用Python
   python3 -m http.server 8000
   
   # 或使用Node.js
   npx http-server -p 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

### 部署到GitHub Pages

1. 克隆本项目到GitHub仓库
2. 进入仓库设置 → Pages
3. 选择分支（通常是main）和文件夹（根目录）
4. 点击"Save"，GitHub会自动部署
5. 几分钟后，您将获得一个公开的URL

## 开发与扩展

### 添加新场景

在 `script.js` 文件中，您可以在 `gameData.scenes` 数组中添加新的诈骗场景：

```javascript
{
    id: "new-scam",
    title: "新诈骗类型",
    description: "场景描述",
    rounds: [
        {
            dialogue: "诈骗信息",
            options: [
                { text: "错误选项1", correct: false, message: "错误提示" },
                { text: "正确选项", correct: true, message: "正确提示" },
                { text: "错误选项2", correct: false, message: "错误提示" }
            ]
        },
        // 更多轮次...
    ],
    winCase: {
        title: "真实案例",
        content: "胜利时显示的真实案例"
    },
    loseCase: {
        title: "真实案例",
        content: "失败时显示的真实案例"
    }
}
```

### 自定义样式

您可以修改 `styles.css` 文件来自定义游戏的外观，包括颜色、字体、布局等。
